# squirrelfmt

A work-in-progress tool to format [Squirrel](http://squirrel-lang.org/) code.

## Acknowledgements

squirrelfmt is heavily inspired by [Inko](https://inko-lang.org/) and
[Full Moon](https://github.com/Kampfkarren/full-moon).
